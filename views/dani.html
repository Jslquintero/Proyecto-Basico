Estructuras Condicionales

Las estructuras condicionales nos permiten ejecutar un bloque de codigo si se cumple una condicion , cuales son las estrucutras condicionales?

  • if
  • else if
  • else

Tenemos la siguiente condicion para mostrar las recetas segun el valor seleccionado del menu desplegable:

  • if (receta == "tacos"){
  • aqui colocaremos el codigo que se ejecutara si la condicion es verdadera
  • }
  • else if (receta == "ensalada"){
  • aqui colocaremos el codigo que se ejecutara si la condicion es verdadera
  • }
  • else{
  • aqui colocaremos el codigo que se ejecutara si la condicion es falsa
  • }
Por lo general else if no es tan utilizado , ya que se podia haber resumido en el uso de dos if o utilizar el else para saber que la condicion seria la contraria.

Operadores logicos

Los operadores logicos nos permiten crear condiciones mas complejas, para ello tenemos los siguientes operadores:

  • && (and) - si ambas condiciones son verdaderas, la condicion sera verdadera
  • || (or) - si una de las condiciones es verdadera, la condicion sera verdadera
  • ! (not) - si la condicion es verdadera, la condicion sera falsa

Por ejemplo, si queremos saber si un numero es mayor a 10 y menor a 20, podemos escribir la siguiente condicion:

  • if (numero > 10 && numero < 20){
  • aqui colocaremos el codigo que se ejecutara si la condicion es verdadera
  • }

Si queremos saber si un numero es mayor a 10 o menor a 5, podemos escribir la siguiente condicion:

  • if (numero > 10 || numero < 5){
  • aqui colocaremos el codigo que se ejecutara si la condicion es verdadera
  • }

Si queremos saber si un numero es mayor a 10, podemos escribir la siguiente condicion:

  • if (numero > 10){
  • aqui colocaremos el codigo que se ejecutara si la condicion es verdadera
  • }

Condicion ternaria

La condicion ternaria nos permite crear una condicion en una sola linea, la cual es muy util cuando queremos asignar un valor a una variable dependiendo de una condicion.

Por ejemplo, si queremos asignar un valor a la variable "numero" dependiendo de si el numero es mayor a 10, podemos escribir la siguiente condicion:

  • numero = (numero > 10) ? 10 : 0;

Si queremos asignar un valor a la variable "numero" dependiendo de si el numero es mayor a 10 o menor a 5, podemos escribir la siguiente condicion:

  • numero = (numero > 10 || numero < 5) ? 10 : 0;

Switch

El switch nos permite crear condiciones mas complejas con valores controlados, esto quiere decir que la estructura no siempre sera la mas adecuada para ciertas ocaciones. , para ello tenemos los siguientes operadores:

  • case - si la condicion es verdadera, se ejecutara el codigo que se encuentra dentro del case
  • break - si la condicion es verdadera, se ejecutara el codigo que se encuentra dentro del case y se saldra del switch
  • default - si ninguna de las condiciones es verdadera, se ejecutara el codigo que se encuentra dentro del default

Por ejemplo, si queremos saber si un numero es mayor a 10 y menor a 20, podemos escribir la siguiente condicion:

  • switch (numero){
  • case 1:
  • aqui colocaremos el codigo que se ejecutara si la condicion es verdadera
  • break;
  • case 2:
  • aqui colocaremos el codigo que se ejecutara si la condicion es verdadera
  • break;
  • default:
  • aqui colocaremos el codigo que se ejecutara si la condicion es falsa
  • break;
  • }

Estructuras Repetitivas

Las estructuras repetitivas nos permiten ejecutar un bloque de codigo varias veces, para ello tenemos los siguientes operadores:

  • while - mientras la condicion sea verdadera, se ejecutara el codigo que se encuentra dentro del while
  • do while - se ejecutara el codigo que se encuentra dentro del do while, y luego se evaluara la condicion
  • for - se ejecutara el codigo que se encuentra dentro del for, y luego se evaluara la condicion
